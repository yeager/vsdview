# VSDView

View Microsoft Visio files (.vsdx/.vsd).
Built-in .vsdx parser, stencils, themes, PDF export.

GTK4/Adwaita 路 Python 路 v0.1.0 路 39 translatable strings

## Install

### Ubuntu / Debian
```bash
sudo apt install vsdview
```
Repo: [yeager/debian-repo](https://github.com/yeager/debian-repo)

### Fedora
```bash
sudo dnf install vsdview
```
Repo: [yeager/rpm-repo](https://github.com/yeager/rpm-repo)
